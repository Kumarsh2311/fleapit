What does Fleapit do?


1. Images and GIFs Searching.

Tired of going to your browser for searching images? Well, With fleapit you can now search image for literally anything. The command for image searching is =image [keyword]. For example, Lets say you want to search image of a snake, you just have to =image snake and the bot will send you an image of a snake. You can also just use =i or =img incase you are in a hurry.

Note:
When you use =image/=img/=i, the images returned will be filtered. So, you don't have to be worried about getting a NSFW image.
We have another image searching command if you want the filter to be off. Use =nsfw [keyword] for unfiltered images. Also, the =nsfw command will only work in a NSFW discord channel.


2. You can search GIFs too.

Need a new discord profile pictute? well you can actually search GIFs using the =gif [keyword] command.
Just type =g tom and it will search a gif with keyword "tom" on Giphy and return you the gif.

3. Search Movie and TV Shows details.

Well, You can server any Movie of TV Show detail using the =movieand =show commands. The results are fetch from TMDb. It will display all the neccessary info that you want to know about a movie or show.

4. Search anything on SPotify.

You can also search for a song by typing =song [song_name]. It will give you the song link from spotify. Beside that you can search an album, an artist's top songs list and an artist info. The commands are =album =artisttop =artist It will show the followers, total overall listeners of am Artist or band.

5. Search a YouTube video.

You can server any youtube video using the =youtube command. It will display the youtube link of that video.
For example, "=youtube blinding lights" will give you the link for the music video of blinding lights.

6. Search a Subreddit feed.

Just type =reddit [subreddit _name] and it will show any recent post fromt the specified subreddit.
For example, "=reddit aww" will send any random post from r/aww.

7. Poll

Type =poll [question] to host a simple yes or no poll. Give the bot 'Manage Messages' permission for better working of this command.

Commands

=help
Display a help message. ex: =help
=image
Search an image. ex: =image dog
=nsfw
Search images with filters off. ex: =nsfw cursed
=movie and =show
Force Movies and TV shows info. ex: =movie Tenet
=artist, =album, =song
Spotify Commands. ex: =artist taylor swift

F.A.Q.

What permissions does this bot need?
"View Channel," "Send Messages," and "Manage messages" are the only permissions this bot needs. "Manage Messages" gives it the permission to delete your poll command after it shows the poll.
